Welcome to Simple Pad

This is a demo notepad app, created to show an understanding of Spring Boot. As you will see, there is not much Javascript implemented, which means its less interactive and everything is being pulled from the servers.

This app was themed using Bootstrap, using PostgreSQL for database storage, and is being hosted on a Heroku server. This is to demonstrate back end development, not to look fancy. Also for optimal experience using the app, try it on a mobile device, its designed for it. Click the button below to:
